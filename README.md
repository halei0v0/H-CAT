### 🌐 主题定位

H-CAT 是基于松坂有希 Mizuki 主题重构的**二次元萌系专属博客模板**，以人气二次元猫猫形象 “猫羽雫” 为核心视觉符号，在完整保留原版 “简约 + 全功能” 架构的基础上，将所有视觉图片资源替换为猫羽雫主题素材，交互按键图标与功能逻辑均保持原版设计，打造兼具实用性与萌系风格的 Astro 博客解决方案。

### 🛠️ 核心功能传承

作为 Mizuki 主题的二次元衍生版，H-CAT 1:1 继承以下核心能力，确保功能无损耗：

- **多场景适配**：支持博客发布、技术文档双模式，PC / 平板 / 移动端响应式布局自动适配，猫羽雫视觉元素在不同设备上均能完整呈现

- **开箱即用体验**：零配置启动基础功能，可自定义主题色（默认匹配猫羽雫形象的柔和色系）、多语言（中 / 英 / 日等），兼顾个性化与易用性

- **全栈功能套件**：内置 Twikoo 评论、PageFind 搜索、Katex 数学公式插件，代码块高亮、图片画廊等功能正常运行，满足技术博主与内容创作者需求

- **内容增强能力**：支持 Markdown 提示容器、 collapsible 折叠模块，插入的猫羽雫图片可自动适配 Photoswipe 画廊，点击查看高清细节

### 🎨 猫羽雫专属视觉体系

相较于原版 Mizuki，H-CAT 以猫羽雫形象重构全量视觉资源，打造沉浸式二次元氛围：

1. **主题视觉符号**：全局 Banner 轮播图替换为猫羽雫不同场景插画（如日常互动、季节主题），页面背景添加低透明度猫羽雫萌系元素，强化主题辨识度

1. **功能组件配图**：友链卡片默认使用猫羽雫风格边框 + 头像，日记模块背景替换为猫羽雫生活场景插画，归档页、 Anime 页图标旁添加猫羽雫小装饰

1. **示例内容升级**：默认博客示例文章配图、About 页个人介绍插图均替换为猫羽雫相关素材，新用户启动即可看到完整主题效果

1. **图标系统兼容**：搜索、评论、代码复制、目录展开等交互按键图标完全沿用原版设计，避免功能认知成本，仅在图标旁添加细微猫羽雫元素点缀（如小爪印）

### 🚀 技术特性保障

- 基于 Astro+Tailwind CSS 构建，静态渲染确保页面加载速度，猫羽雫图片经过优化压缩，不影响性能

- 兼容原版所有命令逻辑，pnpm dev 本地启动、pnpm new-post 创建文章等操作完全一致，迁移成本为零

- 支持自定义图片替换，用户可自行添加更多猫羽雫素材到 src/assets/ 目录，灵活扩展视觉库

### 💡 适用场景

专为二次元爱好者、猫羽雫粉丝、萌系内容创作者设计，尤其适合：

- 二次元主题博客（分享动漫、游戏、周边、技术内容）

- 个人生活记录（搭配猫羽雫插图，让日常分享更具风格）

- 技术博主的个性化展示（用萌系视觉平衡技术内容的严谨感）

- 猫羽雫同好交流平台（友链页、评论区可形成同好互动场景）
